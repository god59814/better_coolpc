# Better CoolPC

更好的原價屋網頁篩選器。從 [原價屋 evaluate.php](https://www.coolpc.com.tw/evaluate.php) 即時抓取組裝清單，提供更直覺的分類、規格與價格篩選，並支援特價整理、菜單籃與參數筆記本。

> 本專案為第三方工具，與原價屋官方無關。資料來源為公開網頁，請合理使用。

## 功能特色

- **類別篩選**：支援全部類別或單一類別瀏覽
- **內建分類**：依原價屋子分類勾選（如主機板晶片組、顯卡系列等）
- **平台篩選**：處理器 / 主機板 / 顯示卡支援 Intel、AMD、NVIDIA 等平台切換
- **品牌篩選**：依類別提供常見品牌下拉選單
- **價格區間**：最低價、最高價數字篩選
- **關鍵字搜尋**：支援商品名稱模糊比對
- **類別專用篩選**
  - 主機板 / 機殼：板型
  - 散熱器 / 水冷：支援腳位
  - 機殼風扇：尺寸、燈效、PWM、反葉
- **數值規格篩選**
  - 顯示卡：長度、風扇數量
  - 機殼：支援顯卡長度、CPU 高度
  - 電源：最小瓦數
  - 散熱器：高度上限
- **就是要白色**：機殼等類別可一鍵篩選白色款式
- **我要撿便宜**：特價模式，依特價期限分組顯示優惠商品
- **菜單籃**：將商品加入清單、調整數量、合併重複項目，資料存於瀏覽器 `localStorage`
- **參數筆記本**：記錄組裝規格（顯卡長度、機殼限制、電源瓦數等），與菜單籃同步，方便比對相容性

## 環境需求

- Python 3.10 以上
- 可連線至原價屋網站

## 安裝

```bash
git clone https://github.com/god59814/better_coolpc.git
cd better_coolpc

pip install -r requirements.txt
```

## 啟動

```bash
python app.py
```

瀏覽器開啟：

```
http://127.0.0.1:5000
```

## 專案結構

```
better_coolpc/
├── app.py              # Flask 網頁入口與路由
├── test.py             # 抓取、解析、篩選邏輯（亦可獨立作 CLI 使用）
├── requirements.txt    # Python 相依套件
└── templates/
    └── index.html      # 前端介面
```

## 使用說明

1. 在頂部選擇**類別**，可按「重置」回到初始狀態。
2. 在**條件**區塊設定品牌、價格、關鍵字與各類專用篩選，按「套用篩選」。
3. 有結果後可按「我要撿便宜」進入特價模式。
4. 商品右側 **+** 可加入菜單籃；右下角綠色按鈕可開啟菜單。
5. 條件區右上角「參數筆記本」可記錄組裝參數，與菜單籃連動。

捲動頁面時，右下角會出現浮動的「我要撿便宜」「清除」按鈕，方便長列表操作。

## 技術說明

- 後端使用 Flask，每次請求從原價屋抓取最新 HTML 並解析 `tbody#tbdy` 商品列
- 解析支援 Big5 / CP950 等繁體中文編碼
- 前端篩選條件透過 GET 參數傳遞，可分享網址保留篩選狀態
- 菜單籃與參數筆記本僅儲存在本機瀏覽器，不會上傳至伺服器

## 注意事項

- 需能正常存取 `coolpc.com.tw`，若原價屋改版可能需調整解析邏輯
- 商品規格由名稱文字推斷，僅供參考，下單前請以原價屋頁面為準
- `app.py` 預設以 `debug=True` 啟動，僅建議本機開發使用
